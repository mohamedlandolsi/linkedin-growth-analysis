# LinkedIn Post Analysis Dashboard

A lightweight, professional Streamlit dashboard for visualizing LinkedIn post performance analysis, sentiment, engagement predictions, and audience ICP scoring.

## ğŸ“Š Dashboard Features

### 1. Post Details
- Full post text display
- Author information
- Extraction metadata (date, browser, status)
- Current engagement metrics (likes, comments, shares, total)

### 2. Post Features Analysis
- Word count and character count
- Hashtags, mentions, emojis, URLs
- Call-to-action (CTA) detection
- Question count
- Lists of all hashtags, mentions, and URLs found

### 3. Sentiment Analysis
- **Gauge Chart**: Visual sentiment score (-1 to +1 scale)
- **Breakdown Chart**: Positive/Neutral/Negative percentage distribution
- Sentiment label (Very Positive, Positive, Neutral, Negative, Very Negative)
- Compound score and confidence level
- Powered by VADER sentiment analyzer (optimized for social media)

### 4. Engagement Prediction
- **Predicted Score**: 0-100 engagement performance score
- **Performance Label**: High Performer, Moderate Performer, or Needs Improvement
- **Percentile Rank**: How the post compares to typical posts
- **Comparison Chart**: Current engagement vs predicted performance
- **Score Breakdown**: Base score + Quality bonus visualization
- **Recommendations**: Actionable tips to improve performance

### 5. Audience ICP Fit
- **Top 10 Most Relevant Audience Members** (ranked by ICP relevance score)
- Audience statistics (total, average relevance, hot/warm leads)
- **Horizontal Bar Chart**: Visual ranking of top audience members
- **Detailed Table**: Name, job title, company, industry, score, priority, match reason
- Color-coded by lead priority (Hot = green, Warm = yellow, Cold = red)

## ğŸ“ Required Data Files

The dashboard automatically loads data from:

1. **Post Data**: `data/json/post_data_sample.json`
   - Scraped post content and engagement metrics
   - Generated by `scripts/linkedin_scraper_simple.py`

2. **Analysis Data**: `data/json/post_analysis.json`
   - Complete analysis results (features, sentiment, prediction)
   - Generated by `scripts/analyze_scraped_post.py`

3. **Audience Ranking** (optional): `data/csv/audience_ranking.csv`
   - ICP-scored audience members
   - Generated by `scripts/export_to_csv.py` with audience data

## ğŸ› Troubleshooting

### Dashboard won't start
```bash
# Clear Streamlit cache
streamlit cache clear

# Restart the dashboard
streamlit run scripts/dashboard.py
```

### Missing data errors
- Ensure you've run the scraper first: `python scripts/linkedin_scraper_simple.py`
- Check that analysis files exist in `data/json/` and `data/csv/`
- Run analysis: `python scripts/analyze_scraped_post.py`

### Port already in use
```bash
# Use a different port
streamlit run scripts/dashboard.py --server.port 8502
```

### Charts not displaying
- Clear browser cache
- Try a different browser
- Check internet connection (Plotly loads some resources online)

## ğŸ”’ Privacy & Security

The dashboard:
- Runs entirely locally on your machine
- Does not send data to external servers (except Streamlit telemetry)
- Does not require LinkedIn API access
- Does not store credentials

To disable Streamlit telemetry:
```bash
# Create/edit ~/.streamlit/config.toml
[browser]
gatherUsageStats = false
```
